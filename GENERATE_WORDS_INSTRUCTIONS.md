# Instructions pour générer les 1500 mots

## Option recommandée : Génération directe par l'IA

Je peux générer les 1500 mots directement ici, mais comme c'est une grande quantité, je vais créer un fichier de données complet.

## Approche proposée

Je vais créer un fichier `data/vocabulary-complete.ts` avec TOUS les 1500 mots organisés par catégories. Ensuite, un script d'import les chargera dans la base de données.

### Étapes

1. **Je génère le fichier avec tous les mots** (1500 mots complets)
2. **Vous lancez le script d'import** qui charge les mots dans la base

### Avantages

- ✅ Pas besoin d'API externe (Gemini)
- ✅ Rapide (import instantané)
- ✅ Contrôle total sur les mots
- ✅ Peut être modifié facilement
